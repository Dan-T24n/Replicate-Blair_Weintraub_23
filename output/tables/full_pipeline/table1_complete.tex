\vspace{1em}

\noindent \textbf{Notes:} Both tables present treatment effects of military policing intervention on crime and security outcomes. Columns (1)-(2) show administrative crime data, columns (3)-(5) show survey-based measures. Treatment refers to direct military policing, spillover refers to blocks within 25 meters of treatment blocks. Confidence intervals in brackets, p-values in parentheses. RI p-values computed using randomization inference with 100,000 simulations.

\vspace{0.3em}

\noindent \textbf{Implementation Note:} Minor differences between reproduction and original (highlighted in \textbf{bold}) occur only in administrative data columns (1-2) which use default standard errors, while survey data columns (3-5) show perfect matches due to clustered standard errors at block level (\texttt{vce(cluster manzana\_code)}). This reflects algorithmic differences between Stata and Python statsmodels in variance-covariance matrix calculations for weighted least squares without clustering. Core treatment effects are identical across all specifications.
